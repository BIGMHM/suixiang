&#12288;&#12288;在本系列的<a href="../../2014/08/privacy-protection-7.md">前一篇博文</a>，已经介绍了各种桌面软件（尤其是国产商业软件）存在的隐私问题。今天这篇介绍一下这些软件的替代品。<b>有些软件比较难以替代，会在本系列的下一篇介绍应对措施。</b><a name='more'></a><!--program-think--><br />&#12288;&#12288;本文中推荐的每一款工具，都附有维基百科的链接。打开维基百科的链接，就可以看到该工具的【官网链接】。<br /><br /><h2>★“首选替代品”和“次选替代品”的差异</h2><br />&#12288;&#12288;在本文中，俺会介绍常见软件类型的各种替代品。针对每一个大类，会同时给出“首选替代品”和“次选替代品”。这两者的差异如下：<br />&#12288;&#12288;<b>首选替代品</b><br />&#12288;&#12288;主要是【国外】非营利机构和组织开发的【开源】软件。<br />&#12288;&#12288;这类软件的特点是――通常情况下，既【没有】咱们朝廷植入的后门，也【没有】欧美政府（比如 NSA）的后门<br /><br />&#12288;&#12288;<b>次选替代品</b><br />&#12288;&#12288;主要是【国外】商业公司开发的【闭源】软件。<br />&#12288;&#12288;这类软件的特点是――通常情况下，【没有】咱们朝廷植入的后门，但【可能有】欧美政府的后门。另外，由于是商业公司开发的，在“隐私保护”方面不如“非营利组织开发的软件”（具体理由参见<a href="../../2013/06/privacy-protection-1.md">本系列第1篇</a>）。<br /><br /><h2>★安全类（防病毒、防木马）</h2><br /><h3>◇不靠谱的软件有哪些？</h3><br />&#12288;&#12288;各种国内商业公司的防病毒、防木马工具，至少包括“奇虎、瑞星、金山、江民、微点、腾讯、百度”等公司的产品。<br /><br /><h3>◇首选替代品</h3><br />&#12288;&#12288;（在首选替代品中列出的，全都是开源项目）<br /><br />&#12288;&#12288;1. <b>改用 Linux，无需安装“防病毒/防木马”软件</b><br />&#12288;&#12288;如果你还在用 Windows，并且饱受病毒和木马的困扰，是时候考虑换成 Linux 啦。如今的 Linux 已经很成熟了，主流的发行版对硬件的支持足够好，而且 Linux 上有一款神器――Wine――它可以让你在 Linux 下运行 Windows 的软件，估计 90% 以上的常用软件（包括“翻墙代理、游戏、媒体播放”等）它都支持。就算碰到少数不支持的，咱们还可以玩【操作系统虚拟机】嘛。没玩过“操作系统虚拟机”的同学，可以看俺博客上的系列扫盲教程，链接在“<a href="../../2012/10/system-vm-0.md">这里</a>”。<br />&#12288;&#12288;由于 Linux 在桌面系统还是小众的，所以病毒和木马的作者都不会考虑 Linux 平台。另外，如果你遵守一些安全规范（比如：不用 root 进行日常操作），基本上就不用担心中病毒和木马。<br />&#12288;&#12288;没用过 Linux 的同学可以看俺之前写的扫盲教程《<a href="../../2013/10/linux-newbie-guide.md">新手如何搞定 Linux 操作系统？</a>》。<br /><br />&#12288;&#12288;2. <b><a href="https://en.wikipedia.org/wiki/Clam_AntiVirus" target="_blank" rel="nofollow">ClamAV</a> / <a href="https://en.wikipedia.org/wiki/ClamWin" target="_blank" rel="nofollow">ClamWin</a></b><br />&#12288;&#12288;防病毒软件中，彻底开源的不多，今天给大伙儿天推荐的 ClamAV 就是一款【<b>彻底开源</b>】的防病毒软件。别看它是开源的，资历比某些商业杀毒软件还要老（诞生于1998年），而且查毒的功能毫不逊色。ClamAV 本身是命令行界面的，ClamWin 提供了 Windows 上的图形界面，内嵌 ClamAV 引擎。<br />&#12288;&#12288;下面列举它的几个特色。<br />特色1――ClamAV 和 ClamWin 性能开销很小，号称是最低功耗的“静音杀毒软件”。<br />特色2――ClamAV 支持的操作系统非常多，除了三大主流的桌面系统，还支持 BSD、Solaris、AIX、HP-UX、OpenVMS、OS/2 等。<br />特色3――ClamWin 支持“便携模式”（做成“绿色软件”），就可以放在“U盘或光盘”上即插即用。官网的说明在“<a href="http://www.clamwin.com/content/view/118/89/" target="_blank" rel="nofollow">这里</a>”。<br />&#12288;&#12288;补充说明：<br />&#12288;&#12288;ClamAV 只有“查毒”功能，没有“杀毒”功能（对已发现的病毒文件，用户可以选择“删除”或“隔离”）。不过这无伤大雅。为啥捏？一旦某个软件感染了病毒，能否【彻底】杀掉，是一个问题？杀掉之后，这个软件是否还能正常工作，又是一个问题。所以俺的经验是：一旦某个系统查出病毒/木马，最安全的处理措施就是――重装系统。综上所述，“杀毒功能”对防病毒软件而言，简直就是鸡肋。而 ClamAV 只提供“查毒”，不提供“杀毒”，反而让它不至于太臃肿。<br />&#12288;&#12288;本文发布后，有热心读者反馈说：ClamAV 的误报率偏高。如果你担心“开源的防病毒”不够好，可以参考“次选替代品”中列举的免费商业软件。<br /><br /><h3>◇次选替代品</h3><br />&#12288;&#12288;（在“次选替代品”中列出的，【不是】开源项目）<br /><br />&#12288;&#12288;下面再介绍几款【国外】的防病毒/防木马产品。因为这些安全产品【不开源】，所以放到“次选替代品”。<br />&#12288;&#12288;考虑到咱们天朝的网民都不喜欢花钱买正版，所以特地介绍几款免费的防病毒和防木马产品（别以为只有 360 是免费滴，国外免费的同类产品也不少哦）。<br /><br />&#12288;&#12288;1. <b><a href="https://zh.wikipedia.org/wiki/Avira_AntiVir" target="_blank" rel="nofollow">Avira / AntiVir（小红伞）</a></b><br />&#12288;&#12288;这款工具来自德国，诞生于1988年，“AntiVir”是之前的名字。它面向家庭用户和非营利组织的版本是免费的。<br />&#12288;&#12288;免费版本支持：Windows、Android、Mac OS X<br />&#12288;&#12288;据俺所知，小红伞在国内用户的口碑不错，所以排第一。<br /><br />&#12288;&#12288;2. <b><a href="https://zh.wikipedia.org/wiki/%E7%A7%91%E6%91%A9%E5%A4%9A%E9%9B%86%E5%9B%A2" target="_blank" rel="nofollow">Comodo（科摩多）</a></b><br />&#12288;&#12288;该公司1998年成立于英国，如今总部在美国。<br />&#12288;&#12288;免费版本支持：Windows、Android、Linux、Mac OS X<br />&#12288;&#12288;Comodo 提供的免费安全工具，种类挺多的（AV、FW、IPS、VPN），所以把它排在第二。<br /><br />&#12288;&#12288;3. <b><a href="https://zh.wikipedia.org/wiki/AVG_Anti-Virus" target="_blank" rel="nofollow">AVG</a></b><br />&#12288;&#12288;这款工具来自捷克，诞生于1997。与小红伞类似，它面向家庭用户和非营利组织的版本是免费的。<br />&#12288;&#12288;免费版本支持：Windows、Android<br /><br />&#12288;&#12288;4. <b><a href="https://zh.wikipedia.org/wiki/Avast!" target="_blank" rel="nofollow">avast（爱维士）</a></b><br />&#12288;&#12288;这款工具跟 AVG 是老乡（也是来自捷克），诞生于1988年。它的免费版本面向家庭用户。<br />&#12288;&#12288;免费版本支持：Windows、Mac OS X<br /><br /><h3>◇备注</h3><br />&#12288;&#12288;如果你对上述列出的防病毒/防木马软件不中意，可以去看维基百科的“<a href="https://en.wikipedia.org/wiki/Comparison_of_antivirus_software" target="_blank" rel="nofollow">这个页面</a>”，里面列出了几十种防病毒/防木马产品的详细对比。<br /><br /><h2>★安全类（主机防火墙）</h2><br /><h3>◇不靠谱的软件有哪些？</h3><br />&#12288;&#12288;各种国内商业公司的主机防火墙，至少包括“奇虎、瑞星、金山、江民”等。<br /><br /><h3>◇替代品</h3><br />&#12288;&#12288;对 Windows 用户而言，你直接用 Windows 自带的防火墙，基本上就能满足需求了。早在 WinXP，就内置了带图形配置界面的防火墙。到了 Vista 之后，内置防火墙的功能又增加了不少。如今，你可以通过 Windows 内置的防火墙实现如下功能：<br />1. 双向过滤（对内流量、对外流量）<br />2. 针对某些端口的过滤<br />3. 针对某些进程的过滤<br />4. ......<br /><br />&#12288;&#12288;对 Linux 用户而言，Linux 内核已经提供了 iptables / nftables，功能足够你用了。其中的 iptables 从 2.4 版本开始整合到 Linux kernel 中。而 nftables 是作为 iptables 的替代品，从今年（2014）1月份开始合并到内核主线。<br /><br /><h3>◇为啥俺推荐系统内置的防火墙？</h3><br />&#12288;&#12288;首先，操作系统内置的防火墙在性能、稳定性、兼容性等方面，至少【不会】比第三方的防火墙更差。<br />&#12288;&#12288;其次，操作系统内置的防火墙，它的开发者也就是操作系统的开发者。所以，【不会】引入额外的隐私风险。<br />&#12288;&#12288;最后，很多商业安全公司（包括国外的），它们提供的“个人主机防火墙”，其宣传的功能大都是唬人的噱头，华而不实，说不定还影响系统的网络性能。<br /><br /><h2>★浏览器</h2><br />&#12288;&#12288;关于“浏览器的选择”，已经在<a href="../../2013/06/privacy-protection-2.md">本系列的第2篇</a>分析过了，此处不再罗嗦。<br /><br /><h2>★输入法</h2><br /><h3>◇不靠谱的软件有哪些？</h3><br />&#12288;&#12288;某些国内商业公司的输入法，至少包括“搜狗拼音、华宇拼音（原“紫光拼音”）、拼音加加、QQ拼音、QQ五笔”等输入法。<br /><br /><h3>◇首选替代品</h3><br />&#12288;&#12288;（在首选替代品中列出的，全都是开源项目）<br /><br />&#12288;&#12288;1. <b><a href="https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%B7%9E%E9%9F%BB%E8%BC%B8%E5%85%A5%E6%B3%95%E5%BC%95%E6%93%8E" target="_blank" rel="nofollow">Rime 输入法（中州韵、小狼毫、鼠须管）</a></b><br />&#12288;&#12288;输入方案：支持十多种输入方案（除了支持多种拼音方案，还包括两种“五笔”，另有一些俺从未听说过滴）。<br /><br />&#12288;&#12288;2. <b><a href="https://zh.wikipedia.org/wiki/FCITX" target="_blank" rel="nofollow">Fcitx /><br />&#12288;&#12288;3. <b><a href="https://zh.wikipedia.org/wiki/IBus" target="_blank" rel="nofollow">iBus</a></b><br />&#12288;&#12288;这也是一个输入法框架，诞生于2008年，原作者是黄鹏。<br />&#12288;&#12288;操作系统支持：类 Unix（Linux、BSD 等）<br />&#12288;&#12288;输入方案：多种拼音、码表（五笔、郑码、仓颉 等）、日文、韩文<br /><br /><h3>◇次选替代品</h3><br />&#12288;&#12288;（在“次选替代品”中列出的，【不是】开源项目）<br /><br />&#12288;&#12288;1. <b><a /><h3>◇不靠谱的软件有哪些？</h3><br />&#12288;&#12288;某些国内商业公司的聊天工具，至少包括“腾讯QQ、网易泡泡、阿里旺旺”等。<br /><br /><h3>◇首选替代品</h3><br />&#12288;&#12288;（在首选替代品中列出的，全都是开源项目）<br /><br />&#12288;&#12288;1. <b><a QQ 的支持需要依靠“pidgin-lwqq”这个插件（此插件也是开源滴，代码库在“<a href="https://github.com/xiehuc/pidgin-lwqq" target="_blank" rel="nofollow">这里</a>”）。据说这个插件利用的是 webQQ 的协议，所以疼逊比较难封杀。<br />&#12288;&#12288;加密支持：有一个 <a rel="nofollow">Pidgin-Encryption</a>”插件。<br /><br />&#12288;&#12288;2. <b><a href="https://zh.wikipedia.org/wiki/Psi_%28%E5%8D%B3%E6%99%82%E9%80%9A%E8%A8%8A%E8%BB%9F%E4%BB%B6%29" target="_blank" 网关跟其它的聊天服务（MSN、AIM/ICQ、Yahoo Messenger 等）互联<br />&#12288;&#12288;加密支持：内置 GnuPG 对消息进行加密；支持 <a href="https://en.wikipedia.org/wiki/Off-the-Record_Messaging" target="_blank" />&#12288;&#12288;这款诞生于2004年，基于 Python 开发。<br />&#12288;&#12288;操作系统支持：由于是采用 Python 语言开发，凡是能运行 Python 的系统都支持。<br />&#12288;&#12288;协议支持：情况跟 Psi 类似。<br />&#12288;&#12288;加密支持：支持 TLS/SSL 和 OpenPGP，支持 <a
